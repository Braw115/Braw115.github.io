第一次搭建个人博客

2018-03-03

创建属于自己的博客首页

前提拥有自己的github账户，新建仓库，仓库名格式为：账户名.github.io并且勾选Initialize this repository with a README选项，在建好的仓库右侧有个settings按钮，点击它，向下拉到GitHub Pages，看到那边有个网址，这个就是你博客的地址，在本地新建名为Blog的目录，clone仓库到本地Blog目录下，这样博客首页就搭建成功了

认识hexo

Hexo是一个快速, 简洁且高效的博客框架. 让上百个页面在几秒内瞬间完成渲染. Hexo支持Github Flavored Markdown的所有功能, 甚至可以整合Octopress的大多数插件. 并自己也拥有强大的插件系统.

安装hexo及初始化

安装

Hexo是基于node.js的, 所以我们在安装它之前需要用到npm安装工具, 这个工具是 node.js 安装包的工具, 所以, 我们先要安装 node.js下载地址http://nodejs.cn/download/ node.js安装成功后nmp也安装成功了，使用如下命令安装hexo

1 npm install -g hexo

初始化

进入Blog目录执行

1 hexo init

生成静态页面

1 hexo g

启动本地服务执行如下命令

1 hexo sever

在浏览器地址栏输入http://localhost:4000看到hexo首页则配置成功

hexo部署

用记事本编辑_config.yml，翻到文本最末尾，将deploy:后面的内容修改如下，其中repository: 后写你的仓库地址 

1  deploy:
2  type: git
3  repository: https://github.com/Braw115/Braw115.github.io.git
4  branch: master

执行如下命令来安装hexo-deployer

1 npm install hexo-deployer-git --save

执行如下代码部署

1 hexo deploy
